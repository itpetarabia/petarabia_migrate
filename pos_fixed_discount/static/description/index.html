POS Fixed Amount Discount

Overview

This module enables a fixed amount discount option along with the percentage.

Main Features:

  • Fixed Amount Global Discount can be applied from the POS screen
  • Fixed Amount Orderline Discount can be applied from the POS screen
  • User can enable or disable the Fixed Amount Global Discount.


Fixed Amount Global Discount enable by activate Global Discounts:



Fixed Amount Global Discount before apply



Fixed Amount Global Discount after apply



Fixed Amount Orderline Discount before apply



Fixed Amount Orderline Discount after apply
