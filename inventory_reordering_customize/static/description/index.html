Internal Transfer Request

Overview

Allows user of one location to make request to another location.

Main Features:

  • Product stock warning message
  • User approval level
  • Scheduled Action for minimum quantity product


Make Warehouse as Source Warehouse

if select Internal transfer source in warehouse, it is taken as the source warehouse in the transfer request. Can manually select destination warehouse


Define Reordering Rules

Inventory > Master Data > Reordering rules. Create Reordering rule for each product, define Warehouse, Minimum and Maximum Quantity


Generating an Internal Transfer Request

Use the 'Transfer Request' menu to create an internal transfer request.


Warning Message

Warning message in transfer request if the product stock exceeds the maximum quantity of the reordering rules or less than the minimum quantity of the reordering rules.


User Approval

Internal transfers require manager level approval


Internal Transfers

Internal transfers will be shown from the Transfer Request. Transfer request is automatically generated when the minimum quantity of the product is reached in the Reordering Rules
